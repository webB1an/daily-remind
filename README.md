# DAILY_REMIND

日常提醒，base on [electron-vite-vue](https://github.com/electron-vite/electron-vite-vue)

## Quick Setup

```sh
# clone the project
git clone https://github.com/webB1an/daily-remind.git

# enter the project directory
cd daily-remind

# install dependency
npm install

# develop
npm run dev

# build for mac
npm run build:mac

# build for win
npm run build:win
```
